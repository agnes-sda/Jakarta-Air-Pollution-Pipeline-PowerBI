## 🛠️ Project Specifications
* **Role:** Data Engineering & Analytics Developer
* **Technology Stack:** Pentaho Spoon (ETL), MySQL (Star Schema Framework), Python, Power BI
* **Core Engineering:** End-to-end data pipelines for data ingestion and deduplication, B-Tree indexing optimization (96.3% response time reduction), and cluster computing exposure.

## Data Engineering Architecture & Workflow
1. **Data Ingestion & ETL:** Utilized **Pentaho Spoon** to build automated graphical ETL pipeline configurations, handling large-scale data cleansing, filtering, and structural transformation.
2. **Database Management:** Loaded the processed datasets into a **MySQL** database structured under a optimized **Star Schema Framework** (Fact and Dimension tables) with B-Tree Indexing.
3. **Data Visualization:** Connected the structured database layer directly to **Microsoft Power BI** to compile interactive, real-time analytical dashboards.
